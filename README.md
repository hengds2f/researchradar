# ResearchRadar

ResearchRadar is an intelligent Academic Paper Discovery and Synthesis Assistant built with Python, Flask, and D3.js. Upload academic PDFs and automatically generate research summaries, identify methodological limitations, or extract key literature findings via section-aware chunk processing!

## Local Setup

1. Install requirements:
`pip install -r requirements.txt`

2. Run the application:
`python app.py`

3. Navigate to `http://localhost:5000` (or `7860` if configured).
